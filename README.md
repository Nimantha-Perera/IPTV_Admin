# IPTV Hotel Admin Panel

## Overview
The **IPTV Hotel Admin Panel** is a web-based admin dashboard built with **Next.js** and **Firebase** that allows administrators to manage IPTV content, live TV channels, on-demand videos, user profiles, and more. The panel provides real-time control over IPTV services, enabling hotel staff to manage guest content, track usage, and update settings efficiently.

## Features

### For Admin Users:
- **Channel Management**: Add, remove, and manage available IPTV channels.
- **Content Management**: Upload and manage on-demand video content such as movies and TV shows.
- **User Management**: View and manage guest profiles, including authentication status and viewing history.
- **Notifications**: Send updates or messages to guests within the app.
- **Analytics & Reports**: View usage analytics and performance metrics for IPTV services.
- **Real-Time Updates**: Instantly update content, channels, and service offerings that sync with the guest app.

## Tech Stack

### Frontend:
- **Next.js**: Framework for building the admin panel with server-side rendering and API routes.
- **React**: For building interactive and dynamic components within the admin panel.
- **Tailwind CSS**: For responsive and modern UI design.
- **Firebase**: For backend services, including authentication, Firestore, and Cloud Storage.

### Other Technologies:
- **Firebase Authentication**: For managing admin user logins and authentication.
- **Cloud Firestore**: For storing and managing data about channels, content, and users.
- **Firebase Cloud Storage**: For uploading and serving media content (videos, images, etc.).
- **Firebase Hosting**: For deploying the admin panel to a live server.
- **NextAuth.js**: For handling authentication with Firebase.
